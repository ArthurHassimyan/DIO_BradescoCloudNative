
# Fundamentos de IA do Microsoft Azure  
## 📚 Módulo: PLN, Inteligência de Documentos e IA Generativa

Este módulo apresenta os principais serviços de inteligência artificial disponíveis no Microsoft Azure para trabalhar com linguagem, documentos e geração de conteúdo. A seguir, você encontrará um resumo claro e direto de cada tecnologia abordada.

---

## 🧠 Processamento de Linguagem Natural (PLN)

O Azure oferece o serviço **AI Language** para entender e trabalhar com textos em linguagem natural. Ele permite:

- ✅ Análise de sentimentos  
- ✅ Extração de entidades (Named Entity Recognition)  
- ✅ Extração de frases-chave  
- ✅ Detecção de idioma  
- ✅ Identificação de informações pessoais (PII/PHI)  
- ✅ Resposta a perguntas com base em contexto

Além disso, o Azure conta com serviços de **fala**, como:

- 🎤 Conversão de fala em texto  
- 🔊 Conversão de texto em fala  
- 🌍 Tradução de voz em tempo real

Esses recursos são fundamentais para **chatbots, assistentes virtuais** e soluções de acessibilidade.

---

## 📄 Inteligência de Documentos

Com o **Azure AI Document Intelligence**, é possível extrair e processar dados de documentos físicos ou digitais:

- 📝 Extração de texto de imagens e PDFs  
- 📊 Análise de tabelas, formulários e documentos estruturados  
- 🧷 Criação de modelos personalizados para reconhecer campos específicos

Esses recursos tornam mais ágil o trabalho com grandes volumes de formulários, contratos e relatórios.

---

## 🔍 Mineração de Conhecimento

Com o **Azure AI Search** e cognição integrada, você pode transformar grandes repositórios de conteúdo em experiências de busca inteligente:

- 📁 Indexação automática de arquivos e documentos  
- 🔎 Extração de informações relevantes  
- ❓ Capacidade de responder perguntas com base no conteúdo processado

Ideal para bases de conhecimento, FAQs e sistemas de suporte.

---

## ✨ IA Generativa

O **Azure OpenAI Service** disponibiliza modelos como o GPT para criar conteúdos novos e inteligentes:

- 📝 Geração de textos (e-mails, artigos, respostas automáticas)  
- 🧠 Resumos automáticos de conteúdos longos  
- 🌐 Tradução avançada e reescrita de textos  
- 💻 Geração de código a partir de linguagem natural

Essas soluções impulsionam produtividade, criatividade e personalização em aplicações modernas.

---

📌 **Resumo:**  
Este módulo reforça como a IA no Azure pode ser aplicada para **compreensão de linguagem**, **extração de dados de documentos** e **criação de conteúdo**, oferecendo ferramentas robustas para transformar dados em valor.
